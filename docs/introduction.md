# This is a fork from [laravel-seo](https://github.com/archtechx/laravel-seo)

## Introduction

@zeus laravel-seo is an extensible SEO meta-tags package for Laravel (forked and maintained by Lara Zeus).

**[Github](https://github.com/lara-zeus/laravel-seo) · [Packagist](https://packagist.org/packages/lara-zeus/laravel-seo)**

## Features

- Manage SEO tags from PHP and Blade
- OpenGraph support out of the box
- Optional Twitter cards, Flipp, and Previewify integration
